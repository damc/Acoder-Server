Acoder is a console application for increasing programmer's productivity by generating code using artificial intelligence (powered by GPT-3 from OpenAI).

Installation

tl;dr - you can install Acoder from Snapstore.

Linux

Ubuntu Software (1st way)

On Ubuntu 20, you can install Acoder by simply opening "Ubuntu Software" and searching for "Acoder".

Snap install (2nd way)

Run:

sudo snap install acoder --classic

If you don't have Snapstore installed already, then see this page for instructions: installing snap store app.

MacOS And Windows

Currently, the application is available only in Snapstore. As far as I know, Snapstore works only with Linux. However, it should be availible to install with Homebrew soon. Check this website after 2 weeks.

Usage

Run in your terminal:

acoder

... and follow the instructions.

Before you can use Acoder, you will need to sign up on this website. After you sign up, you can see an API key which you need to provide to the application (only when you use Acoder for the first time) using the following command:

acoder configure [your API key here]

After that run:

acoder template

This will generate template file "task.md". Open that file and fill the file according to the instructions inside the file.

After that run:

acoder solve

Acoder will display a proposition of changes to the code which you can then accept or reject (or try again).
